Landing page demonstrativa desenvolvida para um escritório de advocacia previdenciária, com foco em apresentação profissional dos serviços, geração de confiança e conversão de visitantes em potenciais clientes.

O projeto reúne informações sobre aposentadorias, benefícios assistenciais, auxílio-doença, pensão por morte, planejamento previdenciário, revisões e mandado de segurança, além de apresentar a profissional responsável e disponibilizar canais diretos de contato.

A página foi construída com HTML5 e CSS3, utilizando layout responsivo, identidade visual personalizada, organização por seções, botões de chamada para ação e integração com WhatsApp e redes sociais.

Este projeto foi desenvolvido para fins de estudo, demonstração técnica e composição de portfólio.
